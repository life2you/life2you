## About | 关于

Hi, I'm `life2you`.

I am a developer with 9 years of Java experience, and in recent years I have
been spending a lot of time building practical Rust tools around terminal UX,
macOS automation, Git workflows, and local AI tooling.

你好，我是 `life2you`。

我有 9 年 Java 开发经验，这几年也在持续投入 Rust 方向，主要做终端体验、
macOS 自动化、Git 工作流，以及偏本地化的 AI 开发工具。

What I care about:

- terminal-first product design
- safe defaults for local tools
- clean and maintainable Rust code
- tools that solve real day-to-day friction

我关注的方向：

- 终端优先的产品体验
- 本地工具的安全默认值
- 易维护、清晰的 Rust 代码
- 真正解决日常痛点的小而美工具

## Focus | 关注方向

- Git workflow tooling for local and team environments
- TUI apps powered by `ratatui` and `crossterm`
- macOS utilities for desktop automation
- developer experience tools around local AI and CLI workflows

- 面向个人与团队环境的 Git 工作流工具
- 基于 `ratatui` 和 `crossterm` 的 TUI 应用
- macOS 桌面自动化工具
- 围绕本地 AI 和 CLI 工作流的开发体验工具
