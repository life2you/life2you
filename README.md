## About

I like building small, focused tools that make local workflows smoother.
Most of my work sits at the intersection of Rust, terminal UX, macOS
automation, and practical developer tooling.

I care about:

- terminal-first product design
- safe defaults for local tools
- clean and maintainable Rust code
- utilities that solve real day-to-day friction

## What I Build

- Git workflow tooling for local and team environments
- TUI apps powered by `ratatui` and `crossterm`
- macOS utilities for desktop automation
- developer experience tools around local AI and CLI workflows

## Current Direction

Right now I am especially interested in tools that feel native to the terminal:

If that sounds like your kind of software, we will probably get along.
